# React App Setup & Run Guide

## 1. Prerequisites

Make sure the following are installed:

- Node.js (recommended version ≥ 18)
Check version:
```bash
node -v
```

- npm or yarn 
```bash
npm -v
```
## 2. Clone the Repository

```bash
git clone https://github.com/username/project-name.git
cd project-name
```
## 3. Install the Dependencies

Using npm:
```bash
npm install
```

Or using yarn:
```bash
yarn install
``` 
## 4. Run the Project (Development Mode)

```bash
npm run dev
```

Then open your browser:
```code
http://localhost:5173
```
## 5. Build for Production

```bash 
npm run preview
```

The production build output will be generated in dist
